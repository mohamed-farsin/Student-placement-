This project is a machine learning-powered Streamlit web app that predicts whether a student is likely to get placed based on their academic performance, skills, and background.

🚀 Features
Predicts placement outcomes using input features like CGPA, 10th & 12th scores, work experience, etc.

Built using Random Forest Classifier

Clean, interactive UI built with Streamlit

Fully scalable and deployable as a web app

Supports custom CSV datasets

📊 Technologies Used
Python

Pandas, NumPy

Scikit-learn

Streamlit (for UI)

LabelEncoder, StandardScaler (for preprocessing)

📁 Input Parameters
Some typical features used in the model:

Gender

10th Percentage (SSC)

12th Percentage (HSC)

Degree Percentage

Work Experience

CGPA

Communication Skills

Target Variable: Placed – Whether the student was placed (1) or not (0)

🧪 Model Training
Trained using a Random Forest Classifier

Achieves good accuracy and generalization on test data

Evaluation metrics: Accuracy, Confusion Matrix, Classification Report
